Video project

A project utilising my favourite media, video and simply embedding it within a website and providing a custom pause / play button.

Project Description

The project involved adding a video into a site, along with it's own pause / play button styling to give the user more control.

Along with a preloader for slower devices, this project was one of my first, and while rather simple, openend my eyes to the importance of a properly responsive webpage.

Challenges

The video proved too bright and needed an opacity filter, also the button's positioning was fine when on a standalone screen with 100% width.

However when I moved into embedding it into this page which had other content, the positioning was absolute and theregore completely out of sync with the rest of the page and it's parent container.

Adding a slider to the button through some transitions proved to be a slightly more arduous task as it was my first approach with moving buttons around, or rather moving around the content on top of them.

Solutions

More of a CSS based task, I had to understand absolute and relatie positioning to place the button where I wanted on both mobile and larger screens.

Using absolute positioning proved to be great for the parent container, but making it relative for the child was when the lighbulb went off. I learned a lot even with just this simple project.

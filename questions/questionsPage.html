Project Description

The project came about from wanting to implement a simple FAQ page, with some embedded javascript to make only the question you click on visible to the user, hiding the other ones.

I always find myself enjoying these simple additions to a site, that really make the user experience more seamless.

Challenges

Really quite a fun little project, the only challenge faced was making adding a toggled class to show the current clicked container and hide the others. Then using a forEach loop to go through all the button containers and display the question.

Solutions

In the JS, which you can find on my github in the footer, I added another method of performing this task, which involved traversing the DOM, selcting all the button containers and looping through them to find the relevent clicked one to display it.

However, in this current implementation I used selectors inside the element to reference the individual question and toggle the show container class that way.

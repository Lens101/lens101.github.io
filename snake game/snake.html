Snake

A childhood classic!

Score :
Controls:
UP, DOWN, LEFT, RIGHT arrow keys to move.
'p' to pause.

Project Description

Written using canvas 2d and vanilla js, making this recreated a small piece of my childhood.

Snake was always dear to me because of it's ubiquity, no matter who's flip phone you were on, it had snake.

Really quite a simple implementation, it involves an edge looper so you don't 'fall off' the map, some collision detection so apples don't spawn inside the snake body and an array to keep track of recent movements to make sure that you haven't bumped into yourself and lost.

Also included is a very simple but i think important functionality... the score! Without it the game doesn't have that addictive feel!

Challenges

The main challenge consisted of figuring our how to stop the apples from spawning in the snakes body which by a default idea of math.random() was occuring.

Subtler problems were figuring our how to pause the canvas on a keypress, essentially pausing the loop if a key had been pressed.

Also the simple problem of using arrow keys to navigate around the canvas, but to not move the website around along with it came up.

Solutions

To stop the apple spawn issue, I kept the math.random() function, but called it again if the apple coordinates existed in the snake's tail array coordinates. If it did, call .random() again, if not, spawn in the apple, simple!

An event listener, looking for a keyboard push for keyCode == 80 ('p'), and a loop wrapping the central game() function determining whether the game was paused using a boolean solved the pausing issue.

The final issue of moving the snake but not the webpage just came about through a very quick look at Stack overflow and a event listener e.preventDefault();

A very fun little project to work on in my free time, I will be doing many more using canvas I can already tell!
